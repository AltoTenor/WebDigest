## Installation
- Clone the repository
```bash
    git clone https://github.com/AltoTenor/WebDigest.git
```
- navigate to project folder
```bash
    cd WebDigest/frontend/
```
- Upload the dist/website-question-answering/ folder to chrome extensions 
- Done.
